# CV Project Description

**Protein Structure Feature and Chain-level Annotation Pipeline**  
Developed a reproducible Python/Linux pipeline for protein structure feature extraction, chain-level annotation, UniProt/SIFTS mapping, quality control, and problem-chain repair. The workflow parses PDB/mmCIF structures, builds entry- and chain-level JSON records, computes local structural and sequence features, integrates optional disorder and Q-score validation tracks, performs UniProt/SIFTS annotation mapping with alignment-based fallback, constructs annotation groups, and generates final QC summaries. The project includes Slurm templates for HPC batch processing and demonstrates practical skills in structural bioinformatics, computational biology, Python scripting, Linux workflows, JSON/CSV data engineering, and reproducible pipeline design.
